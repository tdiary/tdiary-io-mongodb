# TDiary::IO::MongoDB

mongoid adapter for tDiary

## Installation

Add this line to your tDiary's Gemfile.local:

    gem 'tdiary-io-mongodb'

And then execute:

    $ bundle

## Usage

Add follow snipet to your tdiary.conf

```ruby
@io_class = TDiary::IO::MongoDB
```
